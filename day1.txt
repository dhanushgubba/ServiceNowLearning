Lists:
------
* It displays the set of records from a table.
* We can identify the data in form of rows and columns
* Each row --> one record 
* Each column --> one field

* Views enable users to quickly display the same list in multiple ways.
    List controles -> Menu
* If the view is anything other than Default view, the view name appears in bracket
    Eg: View: Mobile

A List that is displayed to user will be sorted by one of the following:
1. Order field
2. Number field
3. Name field

List Personalization:
---------------------
* Personalize list should be used for temporary situations.
* The Personalize list modifies the layout of list of an individual user but it doesnot effect entire Platform users.
    The following can be done to Personalize list:
    1. Add or remove columns
    2. Rearrange columns
    3. Reset column Headers

List Editing:
-------------
* It is a type in which we can edit the data without opening the form
  Steps:
  1. select the field
  2. make the change in the data
  3. click on (tick) mark beside it or can selected multiple records (by shift)
  4. updated

List Filters:
-------------
* It is a set of conditions applied to the table list to isolate a subset of data.
* There are 3 fields of List Filters:
    1. field
    2. Operator
    3. value
* Field operators change depending on data types:
  1. Text value -  is, is not, contains, is one of, starts with, ends with.
  2. Numeric - is, is not, greater than, less than, greater than or is, less than or is.
  3. Date - on, before, after, between, is more than, is less than.

* We also have two buttons AND,OR buttons we can select them and click on OK.

Quick Filters:
--------------
* Right-click on any field --> Show matching Records
* Filter Out --> It allows us to see the only record other than seeing what we don't need

Tags:
-----
* Tags are the easy way to categorize, locate the data
* Right-click on any field -> Assign Tag -> Pre-defined tags / create new tag
* we can also add a tags column using Personalized List Column 
    -- By doing so, we can use in-line editing to add or remove tags from the list view
    -- To save, Press Enter.
